Run Chaser is an app that allows you to run between two points in a selected time of yours.

Choose two points as your endpoints, A and B.
One run is running from any one of the points to the other.
A to B = 1 run
B to A = 1 run

✅ The app gives you a random total number of runs to perform as your target.
✅ You can set your preferred number of runs in the settings as well.
✅ The total number of runs is divided into sets and you get to rest between the sets.
✅ A set of runs can range from one to six runs.
✅ You can set the time per run and the time to rest, in seconds.
✅ You can measure the distance between points A and B and enter it in the settings.

Once settings saved, you are ready to go!
Press START and begin your run chase!

Use this app according to your fitness level and at your own risk.
